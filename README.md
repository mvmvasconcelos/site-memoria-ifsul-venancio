# Memoria

Este é um projeto de website chamado "Memoria". O projeto contém as seguintes páginas:

- Página inicial (`index.html`): Esta página é a página de entrada do website. Ela contém a estrutura HTML e o conteúdo para a página inicial em Português Brasileiro (pt-br).

- Página de linha do tempo (`timeline.html`): Esta página é a página de linha do tempo do website. Ela contém a estrutura HTML e o conteúdo para a página de linha do tempo em Português Brasileiro (pt-br).

- Página de contato (`contact.html`): Esta página é a página de contato do website. Ela contém a estrutura HTML e o conteúdo para a página de contato em Português Brasileiro (pt-br).

O projeto também inclui os seguintes arquivos:

- Arquivo de estilos CSS (`css/styles.css`): Este arquivo contém os estilos CSS para o website. Ele é usado para estilizar os elementos HTML na página inicial, página de linha do tempo e página de contato.

- Arquivo de scripts JavaScript (`js/scripts.js`): Este arquivo contém o código JavaScript para o website. Ele é usado para adicionar interatividade e funcionalidade à página inicial, página de linha do tempo e página de contato.

- Arquivo de configuração do npm (`package.json`): Este arquivo é o arquivo de configuração para o npm. Ele lista as dependências e scripts para o projeto.

## Como executar o projeto

1. Certifique-se de ter o Node.js instalado em sua máquina.

2. Clone este repositório.

3. Navegue até o diretório raiz do projeto.

4. Execute o comando `npm install` para instalar as dependências.

5. Execute o comando `npm start` para iniciar o servidor local.

6. Abra o navegador e acesse `http://localhost:3000` para visualizar o website.
